## Development

Clonar repositorio e instalar dependencias:

```sh
git clone https://github.com/IIC2513-2017-1/pioller-react.git
cd pioller-react
yarn install
```

Iniciar desarrollo:

```sh
yarn start
```

## Producción

Crear versión minificada y ofuscada en `./build`:
```sh
yarn run build
```
